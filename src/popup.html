Popup - 弹出消息框

Popup - 弹出消息框是一个JQuery插件，可以用来代替 DOM 的 title 属性。

Popup - 文件引入方法

方法一

直接引入三个文件，popup.css、jquery.js、popup.js

<!--  步骤一：在 head 中引入 popup.css -->
<link type="text/css" rel="stylesheet" href="popup.css">

<!--  步骤二：引入 JQuery -->
<script type="text/javascript" src="jquery.js"></script>

<!--  步骤三：引入 popup.js，确保JQuery在此之前已经引入 -->
<script type="text/javascript" src="popup.js"></script>

方法二

使用 requareJS 加载，如果您的项目应用了 requareJS，建议使用此方法加载

//  配置路径
require.config({
    paths: {
        jquery  :   'jquery.min',
        popup  :   'popup'
    }
});

//  按照依赖关系加载 popup
require(['jquery','popup'],function($,popup){
    //  --
})

popup - 使用方法

示例一

请将鼠标移到示例上面查看更多信息。

HTML 代码：

<abbr class="ititle" data-content="这是一个示例"/>示例</abbr/>

JavaScript 代码：

$(".ititle").popup();

示例二

HTML 代码：

<button class="ititle-top" data-content="向上弹出">上</button>

<button class="ititle-right" data-content="向右弹出">右</button>

<button class="ititle-bottom" data-content="向下弹出">下</button>

<button class="ititle-left" data-content="向左弹出">左</button>

JavaScript 代码：

/*
 ** 插件名称：popup
 *
 *  语法一 - $(selector).popup([position]);
 *
 ** 参数配置：
 *      position -  弹出消息框的位置，默认为自动计算，可选值为：left,right,top,bottom;
 *
 *  语法二 - $(selector).popup([fn]);
 *
 ** 参数配置：
 *      fn - fn(title,content),接收title和content参数，return一个DOM字符串。
 *
 *  语法三 - $(selector).popup([position],[fn]);
 *
 ** 属性配置：
 *      element data-title   属性 -   消息标题,非必写，可以省略
 *      element data-content 属性 -   消息内容
 *      element data-style   属性 -   消息框自定义样式名称,非必写，可以省略
 *
*/
    $(".ititle-top").popup("top");
    $(".ititle-right").popup("right");
    $(".ititle-bottom").popup("bottom");
    $(".ititle-left").popup("left");

示例三

带有标题的消息提示框。

HTML 代码：

<abbr class="ititle" data-title="示例三" data-content="这是第三个示例，带有标题的消息提示框。">标题</abbr>

JavaScript 代码：

$(".ititle").popup(true);

示例四

改变消息框的默认样式。Popup提供了四种样式，默认为黑色，可选的分别为：popup-white、popup-yellow、popup-red

HTML 代码：

<abbr class="ititle" data-style="popup-white" data-content="白色弹出消息框">popup-white</abbr>

    <abbr class="ititle" data-style="popup-yellow" data-content="黄色弹出消息框">popup-yellow</abbr>

    <abbr class="ititle" data-style="popup-red" data-content="红色弹出消息框">popup-red</abbr>

JavaScript 代码：

$(".ititle").popup();
