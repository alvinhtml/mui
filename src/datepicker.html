DatePicker - 日期时间选择器

DatePicker 是一个用于日期时间选择的JQuery插件，可以选择年月日和时间，内置中英两套模板，兼容主流的HTML5现代浏览器及IE8以上版本。本插件压缩后只有几kb,且使用方便。

DatePicker - 文件引入方法

方法一

直接引入三个文件，datepicker.css、jquery.js、datepicker.js

<!--  步骤一：在 head 中引入 datepicker.css -->
<link type="text/css" rel="stylesheet" href="datepicker.css">

<!--  步骤二：引入 JQuery -->
<script type="text/javascript" src="jquery.js"></script>

<!--  步骤二：引入 datepicker.js，确保JQuery在此之前已经引入 -->
<script type="text/javascript" src="datepicker.js"></script>

方法二

使用 requareJS 加载，如果您的项目应用了 requareJS，建议使用此方法加载

//配置路径
require.config({
    paths: {
        jquery  :   'jquery.min',
        datepicker  :   'datepicker'
    }
});

//按照依赖关系加载 datepicker
require(['jquery','datepicker'],function($,datepicker){
    //  --
})

DatePicker - 使用方法

只选择日期

HTML 代码：

<input type="text" id="dateYear" placeholder="2015-01-01" value="" />

JavaScript 代码：

$("#dateYear").datepicker();

选择日期和时间

HTML 代码：

<input type="text" data-lang="en-us" id="date" placeholder="" value="" />

JavaScript 代码：

/*
 *@ 插件名称：datepicker
 *@ 语法 - $(inputSelector).datepicker(time,callback);
 *@ 参数配置：
 *      time -  布尔型，默认只选定日期， time 为 ture 时，选定日期和时间;
 *      callback -  选择时期时间后的回调函数
 *@ 属性配置
 *      Input data-lang 属性， 可选，定义模板语言，默认为中文
 *          data-lang = "zh-cn" -   中文
 *          data-lang = "en-us" -   英文
*/
$("#date").datepicker(true,"en-us",function(date){
    alert("选定时间是：" + date);
});
